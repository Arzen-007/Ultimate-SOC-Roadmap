# 🛡️ Ultimate SOC Analyst Roadmap (2025/2026 Edition)

Welcome to the **Ultimate SOC Analyst Roadmap**. This repository is designed to guide you from an absolute beginner to an advanced Security Operations Center (SOC) professional. Whether you are looking for a completely free path or a professional certification track, this guide has you covered.

---

## 📈 Learning Path Overview

| Level | Core Skills | Primary Goal |
| :--- | :--- | :--- |
| **Basic (Tier 1)** | Networking, OS Fundamentals, Security Principles, Log Analysis | Triage and basic incident identification |
| **Intermediate (Tier 2)** | Incident Response, Threat Intelligence, Digital Forensics, SIEM/EDR | Deep analysis and containment |
| **Advanced (Tier 3)** | Threat Hunting, Malware Analysis, Automation, Security Architecture | Proactive defense and strategic leadership |

## 🆓 100% Free Roadmap (No Cost)

If you are on a zero-budget, follow this path to build your foundation and earn free certificates.

### 1. Foundations
- **Cisco Skills For All:** [Junior Cybersecurity Analyst Career Path](https://skillsforall.com/) (Free Training + Certificate)
- **ISC2:** [Certified in Cybersecurity (CC)](https://www.isc2.org/landing/1mcc) (Free Training + Exam for 1M people)

### 2. Networking & OS
- **Professors Messer:** [CompTIA Network+ Training](https://www.professormesser.com/) (Free Videos)
- **Cisco NetAcad:** [Introduction to Cybersecurity](https://www.netacad.com/)

### 3. SOC Specific Training
- **TryHackMe:** [SOC Level 1 Path](https://tryhackme.com/path/outline/soclevel1) (Free Tier available)

- **Fortinet:** [NSE 1, 2, and 3](https://training.fortinet.com/) (Free Certificates)

## 💰 Paid Roadmap (Budget to Premium)

If you can invest, these certifications are highly valued by recruiters.

### 🥉 Budget Friendly ($2 - $50)
- **RedTeamLeaders:** [Certified Cyber Security Junior Practitioner (CCSJP)](https://courses.redteamleaders.com/catalog) - **~$2**
- **RedTeamLeaders:** [Certified Application Security Professional (CASP)](https://courses.redteamleaders.com/catalog) - **~$14**
- **Coursera:** [Google Cybersecurity Professional Certificate](https://www.coursera.org/professional-certificates/google-cybersecurity) (Financial aid available)

### 🥈 Professional Tier ($200 - $600)
- **Blue Team Level 1 (BTL1):** [Security Blue Team](https://securityblue.team/why-btl1/) - **~$400**
- **CompTIA CySA+:** [Cybersecurity Analyst](https://www.comptia.org/certifications/cybersecurity-analyst) - **~$400**

### 🥇 Premium/Elite Tier ($1000+)
- **SANS GIAC (GCIA/GCIH):** [GIAC Certified Intrusion Analyst](https://www.giac.org/certification/certified-intrusion-analyst-gcia) - **$8000+**
